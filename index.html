지원 현황 추적

회사명, 직무, 지원일, 마감일, 메모까지. 7가지 진행 상태로 자동 분류되고, 필터로 원하는 항목만 골라봅니다.

  • ✓ 위시리스트 / 지원완료 / 서류심사 / 면접 / 최종합격 / 불합격 / 취소
  • ✓ 회사·직무 무제한 추가
  • ✓ 모바일에서도 한 손 조작

스마트 캘린더

면접·마감일·팔로우업을 색깔로 구분. 날짜 클릭 한 번이면 일정 추가. 다음 일정이 무엇인지 한눈에 보입니다.

  • ✓ 면접 / 마감 / 팔로우업 / 기타 4가지 유형
  • ✓ 미팅 URL · 장소 · 메모 보관
  • ✓ 월간 한눈 뷰

문서함 + 폴더

이력서, 자기소개서, 포트폴리오를 폴더별로 정리. 회사별로 다른 버전을 관리하기에 딱 좋아요.

  • ✓ PDF / PNG / JPG 업로드 (10MB)
  • ✓ 노션 페이지 링크 연결 — 노션에 작성한 자료 그대로 가져오기
  • ✓ 이모지 폴더로 직관적 분류
  • ✓ 본인만 접근 — 안전한 비공개 저장

AI 피드백 출시 예정

이력서를 업로드하면 Claude가 직무별 강점과 보완점을 분석. 적합 직무 추천, 섹션별 리뷰, 커리어 경로 제안까지.

  • ✓ 또래 대비 돋보이는 영역 / 채워가면 좋은 영역
  • ✓ Best Match 직무 + 추천 회사
  • ✓ 단기/중기/장기 커리어 경로

FAQ

자주 묻는 질문

진짜 무료인가요? ▾

2026년 5월 기준, 커리업은 베타 서비스로 출시되어 모든 기능을 무료로 제공하고 있습니다. 결제 정보도, 광고도 없어요.

제 데이터는 안전한가요? ▾

모든 데이터는 본인 계정으로만 접근 가능합니다 (Row Level Security). 다른 유저는 물론 운영자도 권한 없이 들여다볼 수 없도록 분리되어 있어요.

모바일에서도 잘 되나요? ▾

네, 모바일·태블릿·데스크탑 모두 최적화되어 있습니다. 면접장 가는 길에 마지막 메모 확인하기에 좋아요.

기존 노션/엑셀 데이터를 옮길 수 있나요? ▾

노션 페이지는 문서함 → 문서 추가 → 노션 링크에서 URL을 붙여넣으면 바로 연결됩니다. 클릭하면 노션이 새 탭에서 열려요. 엑셀 일괄 가져오기(CSV)는 다음 업데이트에서 추가될 예정입니다.

AI 피드백은 언제 쓸 수 있나요? ▾

현재 개발 중이며, 곧 베타 유저에게 우선 공개할 예정입니다. 가입해두시면 출시 시 가장 먼저 안내드려요.

취준, 더 이상 외롭지 않게.

오늘 가입하고 첫 지원부터 기록해보세요.
